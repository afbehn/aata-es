    Sea \(F\) un cuerpo y \(f(x) \in F[x]\text{.}\) Entonces \(f(x)\) es separable si y solo si \(f(x)\) y \(f'(x)\) son relativamente primos.

    Demostraci√≥n.

    Sea \(f(x)\) separable. Entonces \(f(x)\) se factoriza sobre alg√∫n cuerpo de extensi√≥n de \(F\) como \(f(x) = (x - \alpha_1) (x - \alpha_2) \cdots (x - \alpha_n)\text{,}\) con \(\alpha_i \neq \alpha_j\) para \(i \neq j\text{.}\) Tomando la derivada de \(f(x)\text{,}\) vemos que
    \begin{align*} f'(x) & = (x - \alpha_2) \cdots (x - \alpha_n)\\ & + (x - \alpha_1) (x - \alpha_3) \cdots (x - \alpha_n)\\ & + \cdots + (x - \alpha_1) \cdots (x - \alpha_{n - 1})\text{.} \end{align*}
    Luego, \(f(x)\) y \(f'(x)\) no pueden tener ning√∫n factor com√∫n.
    Para demostrar el rec√≠proco, mostraremos que se cumple la afirmaci√≥n contrapositiva. Supongamos que \(f(x) = (x - \alpha)^k g(x)\text{,}\) con \(k \gt 1\text{.}\) Derivando, tenemos
    \begin{equation*} f'(x) = k ( x - \alpha)^{k-1} g(x) + (x- \alpha)^k g'(x)\text{.} \end{equation*}
    Por lo tanto, \(f(x)\) y \(f'(x)\) tienen un factor com√∫n.

    Teorema 22.1.6.

    Para cada primo \(p\) y para cada entero positivo \(n\text{,}\) existe un cuerpo finito \(F\) con \(p^n\) elementos. M√°s a√∫n, cualquier cuerpo de orden \(p^n\) es isomorfo al cuerpo de descomposici√≥n de \(x^{p^n} -x\) sobre \({\mathbb Z}_p\text{.}\)

    Demostraci√≥n.

    Sea \(f(x) = x^{p^n} - x\) y sea \(F\) el cuerpo de descomposici√≥n de \(f(x)\text{.}\) Por el Lema¬†22.1.5, \(f(x)\) tiene \(p^n\) ceros distintos en \(F\text{,}\) pues \(f'(x) = p^n x^{p^n - 1} - 1 = -1\) es relativamente primo con \(f(x)\text{.}\) Afirmamos que las ra√≠ces de \(f(x)\) forman un subcuerpo de \(F\text{.}\) Ciertamente 0 y 1 son ceros de \(f(x)\text{.}\) Si \(\alpha\) y \(\beta\) son ceros de \(f(x)\text{,}\) entonces \(\alpha + \beta\) y \(\alpha \beta\) tambi√©n son ceros de \(f(x)\text{,}\) pues \(\alpha^{p^n} + \beta^{p^n} = (\alpha + \beta)^{p^n}\) y \(\alpha^{p^n} \beta^{p^n} = (\alpha \beta)^{p^n}\text{.}\) Tambi√©n debemos mostrar que el inverso aditivo y el inverso multiplicativo de cada ra√≠z de \(f(x)\) son ra√≠ces de \(f(x)\text{.}\) Para cualquier cero \(\alpha\) de \(f(x)\text{,}\) sabemos que \(-\alpha\) tambi√©n es cero de \(f(x)\text{,}\) pues
    \begin{equation*} f(-\alpha) = (-\alpha)^{p^n} - (-\alpha) = -\alpha^{p^n} + \alpha = -(\alpha^{p^n} - \alpha) = 0\text{,} \end{equation*}
    suponiendo que \(p\) is impar. Si \(p = 2\text{,}\) entonces
    \begin{equation*} f(-\alpha) = (-\alpha)^{2^n} - (-\alpha) = \alpha + \alpha = 0\text{.} \end{equation*}
    Si \(\alpha \neq 0\text{,}\) entonces \((\alpha^{-1})^{p^n} = (\alpha^{p^n})^{-1} = \alpha^{-1}\text{.}\) Como los ceros de \(f(x)\) forman un subcuerpo de \(F\) y \(f(x)\) se descompone en este subcuerpo, el subcuerpo debe ser todo \(F\text{.}\)
    Sea \(E\) cualquier otro cuerpo de orden \(p^n\text{.}\) Para mostrar que \(E\) es isomorfo a \(F\text{,}\) debemos mostrar que todo elemento en \(E\) es una ra√≠z de \(f(x)\text{.}\) Claramente 0 y 1 son ra√≠ces de \(f(x)\text{.}\) Sea \(\alpha\) un elemento no nulo de \(E\text{.}\) El orden del grupo multiplicativo de elementos no nulos de \(E\) es \(p^n-1\text{;}\) luego, \(\alpha^{p^n-1} =1\) y \(\alpha^{p^n} -\alpha = 0\text{.}\) Como \(E\) contiene \(p^n\) elementos, \(E\) debe ser un cuerpo de descomposici√≥n de \(f(x)\text{;}\) pero, por el Corolario¬†21.2.8, el cuerpo de descomposici√≥n de cualquier polinomio es √∫nico salvo isomorf√≠a.
    El √∫nico cuerpo con \(p^n\) elementos se llama cuerpo de Galois de orden \(p^n\text{.}\) Denotaremos este cuerpo por \(\gf(p^n)\text{.}\)

    Teorema 22.1.7.

    Todo subcuerpo del cuerpo de Galois \(\gf(p^n)\) tiene \(p^m\) elementos, con \(m\) un divisor de \(n\text{.}\) Rec√≠procamente, si \(m \mid n\) para \(m \gt 0\text{,}\) entonces existe un √∫nico subcuerpo de \(\gf(p^n)\) isomorfo a \(\gf(p^m)\text{.}\)

    Demostraci√≥n.

    Sea \(F\) un subcuerpo de \(E = \gf(p^n)\text{.}\) Entonces \(F\) debe ser una extensi√≥n de \(K\) que contiene \(p^m\) elementos, donde \(K\) es isomorfo a \({\mathbb Z}_p\text{.}\) Entonces \(m \mid n\text{,}\) pues \([E:K] = [E:F][F:K]\text{.}\)
    Para demostrar el rec√≠proco, supongamos que \(m \mid n\) para alg√∫n \(m \gt 0\text{.}\) Entonces \(p^m -1\) divide a \(p^n -1\text{.}\) En consecuencia, \(x^{p^m -1} - 1\) divide a \(x^{p^n -1} -1\text{.}\) Por lo tanto, \(x^{p^m} - x\) debe dividir a \(x^{p^n} - x\text{,}\) y todo cero de \(x^{p^m} - x\) tambi√©n es un cero de \(x^{p^n} - x\text{.}\) Luego, \(\gf(p^n)\) contiene, como subcuerpo, un cuerpo de descomposici√≥n de \(x^{p^m} - x\text{,}\) que debe ser isomorfo a \(\gf(p^m)\text{.}\)

    Teorema 22.1.10.

    Si \(G\) es un subgrupo finito de \(F^\ast\text{,}\) el grupo multiplicativo de elementos no nulos de un cuerpo \(F\text{,}\) entonces \(G\) es c√≠clico.

    donde \(n = p_1^{e_1} \cdots p_k^{e_k}\) y los \(p_1, \ldots, p_k\) son primos (no necesariamente distintos). Sea \(m\) el m√≠nimo com√∫n m√∫ltiplo de \(p_1^{e_1}, \ldots, p_k^{e_k}\text{.}\) Entonces \(G\) contiene un elemento de orden \(m\text{.}\) Como todo \(\alpha\) en \(G\) satisface \(x^r - 1\) para alg√∫n \(r\) que divide a \(m\text{,}\) \(\alpha\) debe tambi√©n ser ra√≠z de \(x^m - 1\text{.}\) Como \(x^m -1\) tiene a lo m√°s \(m\) ra√≠ces en \(F\text{,}\) \(n \leq m\text{.}\) Por otra parte, sabemos que \(m \leq |G|\text{;}\) por lo tanto, \(m = n\text{.}\) Luego, \(G\) contiene un elemento de orden \(n\) y tiene que ser c√≠clico.
