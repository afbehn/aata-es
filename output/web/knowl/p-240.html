Párrafo

Supongamos que queremos demostrar que
\begin{equation*} 1 + 2 + \cdots + n = \frac{n(n + 1)}{2} \end{equation*}
para cualquier número natural \(n\text{.}\) Esta fórmula se puede verificar fácilmente para números pequeños tales como \(n = 1\text{,}\) \(2\text{,}\) \(3\text{,}\) or \(4\text{,}\) pero es imposible de verificar para todos los número naturales uno por uno. Para demostrar que la fórmula es verdadera en general, se requiere un método más genérico.
