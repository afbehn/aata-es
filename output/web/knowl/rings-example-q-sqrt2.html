Ejemplo 16.2.3.

El conjunto \({\mathbb Q}( \sqrt{2}\, ) = \{ a + b \sqrt{2} : a, b \in {\mathbb Q} \}\) es un cuerpo. El inverso de un elemento \(a + b \sqrt{2}\) en \({\mathbb Q}( \sqrt{2}\, )\) es
\begin{equation*} \frac{a}{a^2 - 2 b^2} +\frac{- b}{ a^2 - 2 b^2} \sqrt{2}\text{.} \end{equation*}
