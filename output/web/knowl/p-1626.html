Párrafo

Una vez que concebimos grupos isomorfos como el “mismo”, o “fundamentalmente iguales,” o “estructuralmente idénticos,” es natural preguntarnos cuántos grupos finitos “realmente diferentes” existen. El Corolario 9.1.9 nos entrega una respuesta parcial: para cada número primo hay exactamente un grupo finito, con \({\mathbb Z}_p\) como una manifestación concreta.
