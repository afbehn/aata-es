Supongamos que nuestro mensaje original es un \(0\) o un \(1\text{,}\) y que \(0\) se codifica como \((000)\) y \(1\) se codifica como \((111)\text{.}\) Si solo ocurre un error durante la transmisión, podemos detectar y corregir el error. Por ejemplo, si se recibe un \((101)\text{,}\) el segundo bit debe haberse cambiado de \(1\) a \(0\text{.}\) La palabra clave transmitida originalmente debe haber sido \((111)\text{.}\) Este método detectará y corregirá todos los errores individuales.
En Cuadro 8.1.5, presentamos todas las palabras posibles que podrían recibirse para las palabras clave transmitidas \((000)\) y \((111)\text{.}\) Cuadro 8.1.5 también muestra el número de bits en los que cada tupla \(3\) recibida difiere de cada palabra de código original.
