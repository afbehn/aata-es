Sea \(\phi : R \rightarrow S\) un homomorfismo de anillos.
  1. Si \(R\) es un anillo conmutativo, entonces \(\phi(R)\) es un anillo conmutativo.
  2. \(\phi( 0 ) = 0\text{.}\)
  3. Sean \(1_R\) y \(1_S\) las identidades de \(R\) y \(S\text{,}\) respectivamente. Si \(\phi\) es sobre, entonces \(\phi(1_R) = 1_S\text{.}\)
  4. Si \(R\) es un cuerpo y \(\phi(R) \neq \{ 0 \}\text{,}\) entonces \(\phi(R)\) es un cuerpo.
