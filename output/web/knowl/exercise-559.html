Ejercicio 1.

Este ejercicio ejemplifica el Teorema Teorema 15.2.5. El subgrupo conmutador se puede obtener con el método .commutator(). Para el grupo dihedral de orden \(40\text{,}\) \(D_{20}\) (DihedralGroup(20) en Sage), calcule el subgrupo conmutador y forme el cociente del grupo dihedral por este subgrupo. Compruebe que este cociente es abeliano. ¿Puede identificar a qué grupo conocido es isomorfo este cociente?
