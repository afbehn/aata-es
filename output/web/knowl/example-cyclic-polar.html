Ejemplo 4.2.4.

Supongamos que \(z = 2 \cis 60^{\circ}\text{.}\) Entonces
\begin{equation*} a = 2 \cos 60^{\circ} = 1 \end{equation*}
y
\begin{equation*} b = 2 \sin 60^{\circ} = \sqrt{3}\text{.} \end{equation*}
Luego, la representación cartesiana es \(z = 1+\sqrt{3}\, i\text{.}\)
Recíprocamente, si nos entregan la representación cartesiana de un número complejo, puede ser útil conocer su representación polar. Si \(z = 3 \sqrt{2} - 3 \sqrt{2}\, i\text{,}\) entonces
\begin{equation*} r = \sqrt{a^2 + b^2} = \sqrt{36 } = 6 \end{equation*}
y
\begin{equation*} \theta = \arctan \left( \frac{b}{a} \right) = \arctan( - 1) = 315^{\circ}\text{,} \end{equation*}
así \(3 \sqrt{2} - 3 \sqrt{2}\, i=6 \cis 315^{\circ}\text{.}\)
