Párrafo

Este problema continúa el anterior. Construya el anillo de polinomios sobre \({\mathbb Z}_3\) y en este anillo use \(p(x)\) para generar un ideal principal. Finalmente construya el cociente del anillo de polinomios por este ideal. Como el polinomio es irreducible, este cociente es un cuerpo, y por la Proposición 21.1.12 este cociente es isomorfo al cuerpo de números del ejercicio anterior.
