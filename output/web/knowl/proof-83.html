Demostración

Como \(G\) es un grupo abeliano, tenemos que \(G_i\) es un subgrupo de \(G\) para \(i = 1, \ldots, k\text{.}\) Como la identidad tiene orden \(p_i^0 = 1\text{,}\) sabemos que \(1 \in G_i\text{.}\) Si \(g \in G_i\) tiene orden \(p_i^r\text{,}\) entonces \(g^{-1}\) también debe tener orden \(p_i^r\text{.}\) Finalmente, si \(h \in G_i\) tiene orden \(p_i^s\text{,}\) entonces
\begin{equation*} (gh)^{p_i^t} = g^{p_i^t} h^{p_i^t} = 1 \cdot 1 = 1\text{,} \end{equation*}
donde \(t\) es el mayor entre \(r\) y \(s\text{.}\)
Debemos mostrar que
\begin{equation*} G = G_1 G_2 \cdots G_k \end{equation*}
y \(G_i \cap G_j = \{1 \}\) para \(i \neq j\text{.}\) Supongamos que \(g_1 \in G_1\) está en el subgrupo generado por \(G_2, G_3, \ldots, G_k\text{.}\) Entonces \(g_1 = g_2 g_3 \cdots g_k\) para \(g_i \in G_i\text{.}\) Como \(g_i\) tiene orden \(p_i^{\alpha_i}\text{,}\) sabemos que \(g_i^{p_i^{\alpha_i}} = 1\) para \(i = 2, 3, \ldots, k\text{,}\) y \(g_1^{p_2^{\alpha_2} \cdots p_k^{\alpha_k}} = 1\text{.}\) Como el orden de \(g_1\) es una potencia de \(p_1\) y \(\gcd(p_1, p_2^{\alpha_2} \cdots p_k^{\alpha_k}) = 1\text{,}\) tenemos que \(g_1 = 1\) y la intersección de \(G_1\) con cualquiera de los subgrupos \(G_2, G_3, \ldots, G_k\) es la identidad. Un argumento similar muestra que \(G_i \cap G_j = \{1 \}\) para \(i \neq j\text{.}\)
A continuación, debemos demostrar que es posible escribir cada \(g \in G\) como un producto \(g_1 \cdots g_k\text{,}\) donde \(g_i \in G_i\text{.}\) Dado que el orden de \(g\) divide el orden de \(G\text{,}\) lo sabemos
\begin{equation*} |g| = p_1^{\beta_1} p_2^{\beta_2} \cdots p_k^{\beta_k} \end{equation*}
para ciertos números enteros \(\beta_1, \ldots, \beta_k\text{.}\) Dejando \(a_i = |g| /p_i^{\beta_i}\text{,}\) los \(a_i\) son relativamente primos; por tanto, existen números enteros \(b_1, \ldots, b_k\) such that \(a_1 b_1 + \cdots + a_k b_k = 1\text{.}\) Como consecuencia,
\begin{equation*} g = g^{a_1 b_1 + \cdots + a_k b_k} = g^{a_1 b_1} \cdots g^{a_k b_k}\text{.} \end{equation*}
Como
\begin{equation*} g^{(a_i b_i ) p_i^{\beta_i}} = g^{b_i |g|} = e\text{,} \end{equation*}
deducimos que \(g^{a_i b_i}\) debe estar en \(G_{i}\text{.}\) Sea \(g_i = g^{a_i b_i}\text{.}\) Entonces \(g = g_1 \cdots g_k \in G_1 G_2 \cdots G_k\text{.}\) Por lo tanto, \(G = G_1 G_2 \cdots G_k\) es un producto directo interno de subgrupos.
