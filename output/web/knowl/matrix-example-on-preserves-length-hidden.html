El vector \({\mathbf x} =(3,4)^\transpose\) tiene longitud \(\sqrt{3^2 + 4^2} = 5\text{.}\) Podemos también ver que la matriz ortogonal
\begin{equation*} A= \begin{pmatrix} 3/5 & -4/5 \\ 4/5 & 3/5 \end{pmatrix} \end{equation*}
preserva la longitud de este vector. El vector \(A{\mathbf x} = (-7/5,24/5)^\transpose\) también tiene longitud 5.
