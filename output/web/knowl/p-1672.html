Párrafo
En general, el subgrupo \(n {\mathbb Z}\) de \({\mathbb Z}\) es normal. Las clases laterales de \({\mathbb Z } / n {\mathbb Z}\) son
\begin{gather*} n {\mathbb Z}\\ 1 + n {\mathbb Z}\\ 2 + n {\mathbb Z}\\ \vdots\\ (n-1) + n {\mathbb Z}\text{.} \end{gather*}
La suma de clases laterales \(k + n{\mathbb Z}\) y \(l + n{\mathbb Z}\) es \(k+l + n{\mathbb Z}\text{.}\) Note que hemos escrito las clases laterales de forma aditiva, pues la operación del grupo es la adición de enteros.
