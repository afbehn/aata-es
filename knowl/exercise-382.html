Ejercicio 2.

Considere el grupo cociente del grupo de simetrías de un octógono regular, por el subgrupo cíclico de orden 4 generado por una rotación en un cuarto de vuelta. Use la función coset_product para determinar la tabla de Cayley para este grupo cociente. Use los números de cada clase lateral, resultantes del método .cosets() como nombres para los elementos del grupo cociente. Necesitará construir la tabla “a mano” pues no hay una forma fácil de lograr que los comandos de Sage hagan esto. Puede construir una tabla en el editor del Notebook Sage (shift-click en una línea azul) o puede leer la documentación del método html.table().
