Proposición 20.1.5.

Sea \(V\) un espacio vectorial sobre \(F\text{.}\) Entonces cada uno de los siguiented enunciados es verdadero.
  1. \(0v ={\mathbf 0}\) para todo \(v \in V\text{.}\)
  2. \(\alpha {\mathbf 0} = {\mathbf 0}\) para todo \(\alpha \in F\text{.}\)
  3. Si \(\alpha v = {\mathbf 0}\text{,}\) entonces ya sea \(\alpha = 0\) o \(v = {\mathbf 0}\text{.}\)
  4. \((-1) v = -v\) para todo \(v \in V\text{.}\)
  5. \(-(\alpha v) = (-\alpha)v = \alpha(-v)\) para todo \(\alpha \in F\) y para todo \(v \in V\text{.}\)
