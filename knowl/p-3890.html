Párrafo

La respuesta es usar la estructura de espacio vectorial de la torre aplanada. Como es una extensión de grado \(8\) de los racionales, las primeras \(8\) potencias de un elemento primitivo c forman una base cuando el cuerpo se ve como un espacio vectorial con los racionales como escalares. Es suficiente saber como cada automorfismo de cuerpos actúa en esta base para completamente especificar la definición del automorfismo. Esto es,
\begin{align*} \tau(x)&=\tau\left(\sum_{i=0}^7\,q_ic^i\right)&&q_i\in{\mathbb Q}\\ &=\sum_{i=0}^7\,\tau(q_i)\tau(c^i)&&\tau\text{ es un automorfismo de cuerpos}\\ &=\sum_{i=0}^7\,q_i\tau(c^i)&&\text{los racionales quedan fijos} \end{align*}
Así podemos calcular el valor de un automorfismo de cuerpos en cualquier combinación lineal de potencias del elemento primitivo como combinación lineal de los valores del automorfismo de cuerpos solo en las potencias del elemento primitivo. Esta se conoce como “base de potencias”, lo que podemos obtener simplemente con el método .power_basis(). Empezaremos con un ejemplo de como usar esta base. Ilustraremos con el cuarto automorfismo del grupo de Galois. Note que el método .vector() es conveniente en tanto transforma una combinación lineal de potencias de c en un vector que solo retiene los coeficientes. (Note además que \(\tau\) está completamente definido por el valor de \(\tau(c)\text{,}\) pues como es un automorfismo de cuerpos \(\tau(c^k)=(\tau(c))^k\text{.}\) Sin embargo, igual debemos trabajar con la base de potencias completa para aprovechar la estructura de espacio vectorial.)
