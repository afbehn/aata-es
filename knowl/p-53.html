Sean \(A\text{,}\) \(B\text{,}\) y \(C\) conjuntos. Entonces
  1. \(A \cup A = A\text{,}\) \(A \cap A = A\text{,}\) y \(A \setminus A = \emptyset\text{;}\)
  2. \(A \cup \emptyset = A\) y \(A \cap \emptyset = \emptyset\text{;}\)
  3. \(A \cup (B \cup C) = (A \cup B) \cup C\) y \(A \cap (B \cap C) = (A \cap B) \cap C\text{;}\)
  4. \(A \cup B = B \cup A\) y \(A \cap B = B \cap A\text{;}\)
  5. \(A \cup (B \cap C) = (A \cup B) \cap (A \cup C)\text{;}\)
  6. \(A \cap (B \cup C) = (A \cap B) \cup (A \cap C)\text{.}\)
