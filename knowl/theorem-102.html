Teorema 19.1.13.

Si \(L\) es un reticulado, entonces las operaciones binarias \(\vee\) y \(\wedge\) satisfacen las siguientes propiedades para \(a, b, c \in L\text{.}\)
  1. Leyes conmutativas: \(a \vee b = b \vee a\) y \(a \wedge b = b \wedge a\text{.}\)
  2. Leyes asociativas: \(a \vee ( b \vee c) = (a \vee b) \vee c\) y \(a \wedge (b \wedge c) = (a \wedge b) \wedge c\text{.}\)
  3. Leyes idempotentes: \(a \vee a = a\) y \(a \wedge a = a\text{.}\)
  4. Leyes de absorción: \(a \vee (a \wedge b) = a\) y \(a \wedge ( a \vee b ) =a\text{.}\)
