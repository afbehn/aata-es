Teorema 8.2.3.

Sea \(d_{\min}\) la distancia mínima para un código de grupo \(C\text{.}\) Entonces \(d_{\min}\) es el mínimo de todos los pesos de las palabras no nulas en \(C\text{.}\) Es decir,
\begin{equation*} d_{\min} = \min\{ w({\mathbf x}) : { {\mathbf x} \neq {\mathbf 0} } \}. \end{equation*}
