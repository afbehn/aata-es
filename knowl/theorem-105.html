Teorema 19.2.2.

Un conjunto \(B\) es un álgebra Booleana si y solo si existen operaciones binarias \(\vee\) y \(\wedge\) en \(B\) que satisfacen los siguientes axiomas.
  1. \(a \vee b = b \vee a\) y \(a \wedge b = b \wedge a\) para \(a, b \in B\text{.}\)
  2. \(a \vee ( b \vee c) = (a \vee b) \vee c\) y \(a \wedge ( b \wedge c) = (a \wedge b) \wedge c\) para \(a, b, c \in B\text{.}\)
  3. \(a \wedge ( b \vee c ) = (a \wedge b ) \vee ( a \wedge c )\) y \(a \vee ( b \wedge c ) = (a \vee b ) \wedge ( a \vee c )\) para \(a, b, c \in B\text{.}\)
  4. Existen elementos \(I\) y \(O\) tales que \(a \vee O = a\) y \(a \wedge I = a\) para todo \(a \in B\text{.}\)
  5. Para todo \(a \in B\) existe \(a' \in B\) tal que \(a \vee a' = I\) y \(a \wedge a' = O\text{.}\)
