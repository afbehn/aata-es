La permutación (no trivial) más simple es un ciclo de largo \(2\text{.}\) Tales ciclos se llaman transposiciones. Como
\begin{equation*} (a_1, a_2, \ldots, a_n ) = (a_1, a_n ) (a_1, a_{n-1} ) \cdots ( a_1, a_3 ) (a_1, a_2 )\text{,} \end{equation*}
cualquier ciclo puede ser escrito como el producto de transposiciones, llevándonos a la siguiente proposición.
