Ejercicio 17. Números de Fibonacci.

Los Números de Fibonacci son
\begin{equation*} 1, 1, 2, 3, 5, 8, 13, 21, \ldots\text{.} \end{equation*}
Podemos definirlos recursivamente como \(f_1 = 1\text{,}\) \(f_2 = 1\text{,}\) y \(f_{n + 2} = f_{n + 1} + f_n\) para \(n \in {\mathbb N}\text{.}\)
  1. Demuestre que \(f_n \lt 2^n\text{.}\)
  2. Demuestre que \(f_{n + 1} f_{n - 1} = f^2_n + (-1)^n\text{,}\) \(n \geq 2\text{.}\)
  3. Demuestre que \(f_n = [(1 + \sqrt{5}\, )^n - (1 - \sqrt{5}\, )^n]/ 2^n \sqrt{5}\text{.}\)
  4. Muestre que \(\phi = \lim_{n \rightarrow \infty} f_{n + 1} / f_n = (\sqrt{5} + 1)/2\text{.}\) La constante \(\phi\) se conoce como proporción áurea.
  5. Demuestre que \(f_n\) y \(f_{n + 1}\) son relativamente primos.
