Item 8.6.26.d.

¿Cuál es el número de bits de verificación y el número de bits de información en un código de Hamming de bloque \((m,n)\text{?}\) Encuentre tanto una cota superior como una cota inferior para el número de bits de información en términos del número de bits de verificación. Códigos de Hamming que tengan el máximo posible número de bits de información con \(k\) bits de verificación se llaman perfectos. Cada posible síndrome a excepción de \({\mathbf 0}\) ocurre como una columna. Si el número de bits de información es menor al máximo, entonces el código se llama recortado. En este caso, dé un ejemplo donde algunos síndromes puedan representar errores múltiples.
