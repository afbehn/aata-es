Ejemplo 1.2.1.

Sea \({\mathbb R}\) el conjunto universal y supongamos que
\begin{equation*} A = \{ x \in {\mathbb R} : 0 \lt x \leq 3 \} \quad \text{y} \quad B = \{ x \in {\mathbb R} : 2 \leq x \lt 4 \}\text{.} \end{equation*}
Entonces
\begin{align*} A \cap B & = \{ x \in {\mathbb R} : 2 \leq x \leq 3 \}\\ A \cup B & = \{ x \in {\mathbb R} : 0 \lt x \lt 4 \}\\ A \setminus B & = \{ x \in {\mathbb R} : 0 \lt x \lt 2 \}\\ A' & = \{ x \in {\mathbb R} : x \leq 0 \text{ o } x \gt 3 \}\text{.} \end{align*}
